# Disease Gene Explorer

A Python-based exploratory tool for studying disease-associated genes and their biological relevance.

## Project Overview

Disease-associated gene analysis is an important part of bioinformatics, precision medicine, and translational biomedical research. This project provides a structured way to explore selected genes associated with diseases such as diabetes, kidney disease, Alzheimer's disease, Parkinson's disease, and ALS.

The objective of this project is to organize disease-gene relationships and provide simple programmatic tools for searching, filtering, and summarizing biologically relevant gene information.

## Implemented Features

- Disease-wise gene exploration
- Gene-specific search
- Biological role annotation
- Disease relevance summary
- CSV-based structured dataset
- Python-based analysis workflow
